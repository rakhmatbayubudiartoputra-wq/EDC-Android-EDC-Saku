# EDC-Android-EDC-Saku
Harga : EDC Andorid Rp. 1.999.000 &amp; EDC Saku Rp. 499.000
Kerjasama antara PT Permodalan Nasional Madani (PNM) dengan Buku Agen (BukuWarung) bertujuan untuk mendorong digitalisasi UMKM, khususnya nasabah PNM, melalui pencatatan keuangan digital dan penggunaan layanan keuangan modern.

Berikut adalah poin-poin utama kerjasama PNM dan Buku Agen:
Digitalisasi Pencatatan Keuangan: Buku Agen membantu nasabah PNM (terutama Mekaar dan ULaMM) untuk beralih dari pencatatan manual ke digital. Ini mempermudah nasabah dalam memantau arus kas, utang-piutang, dan keuntungan usaha.
Pelatihan dan Pendampingan: PNM dan Buku Agen mengadakan pelatihan pencatatan keuangan sederhana bagi para pelaku usaha mikro.
Penggunaan Teknologi EDC BukuAgen: Nasabah PNM didorong untuk menggunakan teknologi seperti EDC Saku BukuAgen untuk transaksi keuangan. Ini memudahkan nasabah menerima pembayaran non-tunai dan melakukan transaksi perbankan (transfer, tarik tunai, pembayaran).
Peningkatan Efisiensi: Dengan menggunakan aplikasi Buku Agen, nasabah dapat mengelola usaha dengan lebih terstruktur, meningkatkan inklusi keuangan, dan mempermudah akses modal lanjutan.
